# GREEN-API Test Task

Тестовое задание на позицию **Специалист технической поддержки 2 линии**.

## Описание
HTML-страница для работы с GREEN-API, реализующая вызовы следующих методов:
- getSettings
- getStateInstance
- sendMessage
- sendFileByUrl

## Как использовать
1. Создать инстанс в личном кабинете GREEN-API
2. Подключить WhatsApp, отсканировав QR-код
3. Открыть страницу:
   https://zhusupovdaniyal.github.io/green-api-test/
4. Ввести `idInstance` и `ApiTokenInstance`
5. Нажать кнопки и посмотреть результат в поле Response

## Технологии
- HTML
- JavaScript (Fetch API)
- GREEN-API

## Примечание
Проект является демонстрационным. В реальных условиях токены хранились бы на backend.
